=========================================
HaZardModding Project Identity
=========================================

This document explains which releases, websites,
repositories, and services belong to the official
HaZardModding project.

For the full legal terms see:
hzm_legal.txt


-----------------------------------------
Official Project
-----------------------------------------

The HaZardModding project refers to the cooperative
game modifications developed and maintained by the
HaZardModding team.


-----------------------------------------
Official Names
-----------------------------------------

The following identifiers represent the official project:

 - HaZardModding
 - HZM
 - HaZardModding Coop


-----------------------------------------
Official Project Channels
-----------------------------------------

Official project information and releases are published
only through the following channels:

Website
 - https://hazardmodding.com

GitHub
 - https://github.com/HaZardModding

ModDB
 - https://moddb.com/mods/medal-of-honor-coop-hazardmodding


-----------------------------------------
Unofficial Forks and Derivatives
-----------------------------------------

Forks and modified versions of the project are allowed, but they must:

 - use a different project name
 - clearly state that they are unofficial
 - not present themselves as official HaZardModding releases

Example:

    XYZ Coop Expansion (based on HaZardModding)


-----------------------------------------
Server Operators
-----------------------------------------

Game servers may run modified versions of the mod.

However, servers distributing modified client files must
clearly indicate that those files are unofficial builds.


-----------------------------------------
Reporting Impersonation
-----------------------------------------

If you encounter websites, downloads, or communities
claiming to represent the HaZardModding project but
not listed above, they are not official.

Please report such cases to the project maintainers.
